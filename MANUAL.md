# Manual checklist

Complete these tasks after `script/bootstrap` finishes.

1. Restart the Mac if FileVault, macOS updates, or changed settings require it.
2. Sign in to 1Password, browsers, Google Drive, Google Account for contacts
   and calendar, and any other accounts needed.
3. Move `~/Desktop/FileVault Recovery Key.txt` into 1Password if it exists.
4. Make Brave the default Browser
5. Setup 1Password preferences
  - Configure unlock preferences.
  - Install Browser extension.
  - Enable SSH Agent.
6. Start Alfred and restore license and default shortcut.
7. Fire up nvim to trigger the setup
